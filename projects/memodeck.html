Memodeck

Summary

Date 2018 (one week)
Tech C++

Simple quizz like command line app to memorize the order of a deck of cards.

  • Learn some of the most famous stacked decks, or create your own in simple text files
  • Features different tests like sequence or position matching.
  • Learn in your own language : easily translatable with text files

Now what could a magician possibly use that for, I wonder...
